# GEO Indexing Strategy

GEO means Generative Engine Optimization.

SydTek University pages should be structured for both human learners and AI systems.

Each course page should include:
- title
- description
- learning outcomes
- syllabus
- modules
- assignments
- instructor information
- transcript-rich content
- structured metadata

Primary GEO assets:
- GitHub repositories
- llms.txt
- schema.org JSON-LD
- course metadata
- transcript archives
- syllabus pages
