# Platform Architecture

SydTek University operates through a layered architecture.

## Shopify = Campus

Handles:
- homepage
- admissions
- bookstore
- payments
- product catalog
- membership offers

## Thinkific = Classroom

Handles:
- courses
- lessons
- modules
- quizzes
- certificates
- student progress

## YouTube = Lecture Hall

Handles:
- public lectures
- discovery
- GEO visibility
- student recruitment

## GitHub = Institutional Memory

Handles:
- curriculum documentation
- metadata standards
- repo-based academic archive
- llms.txt
