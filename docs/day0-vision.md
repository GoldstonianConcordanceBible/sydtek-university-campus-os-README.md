# Day 0 Vision

SydTek University is a Shopify-based digital campus designed to deliver a structured university-style learning experience through certificate programs, bookstore infrastructure, LMS classrooms, public lectures, and GEO-native academic metadata.

Phase I focuses on proving the model through a lean stack:

- Shopify as the campus layer
- Thinkific as the classroom layer
- YouTube as the public lecture hall
- GitHub as institutional memory
- Klaviyo as the communication layer

The goal is proof of concept, enrollment, cash flow, and institutional legitimacy.
